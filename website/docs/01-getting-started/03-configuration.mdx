# Configuration

Grats has a few configuration options. They can be set under the `"grats"` key in your in your project's `tsconfig.json` file:

```json title="tsconfig.json"
{
  "grats": {
    // Config options go here. See below:
  },
  "compilerOptions": {
    // ... TypeScript config...
  }
}
```
